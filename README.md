# LangChain Learning

A hands-on exploration of LangChain fundamentals including LLM integration, 
conversation memory, prompt templates, and RAG (Retrieval Augmented Generation).

## What's covered

- Basic LLM calls with LangChain and Groq
- Conversation memory using message history
- System prompts and prompt templates
- Chains using the pipe operator
- Document chunking and text splitting
- Vector storage with ChromaDB
- Semantic retrieval with HuggingFace embeddings
- Complete RAG pipeline

## Tech Stack

- LangChain
- Groq API (Llama 3.3 70B)
- ChromaDB
- HuggingFace Sentence Transformers

## Setup

pip install langchain langchain-groq langchain-community 
langchain-text-splitters chromadb sentence-transformers

Add your GROQ_API_KEY to a .env file.
